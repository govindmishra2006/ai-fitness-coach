# AI Fitness Coach

A full-stack AI-powered fitness application that predicts calories burned using Machine Learning.

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Django
- Django REST Framework

### Machine Learning
- Scikit-learn
- Pandas
- NumPy

## Features

- AI calorie prediction
- Full-stack architecture
- REST API integration
- Machine Learning model deployment
- Responsive modern UI

## Machine Learning Workflow

- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
- Prediction API integration
